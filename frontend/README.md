# 🖥️ AI Tutor – Frontend

This is the React-based UI for:
- Uploading course materials
- Visualizing detected chapters
- Triggering quiz generation
- Interactively answering AI-generated quizzes

## 🧰 Stack

- React + Vite
- TailwindCSS
- Axios for API communication

## ⚙️ Setup

```bash
cd frontend
yarn install
yarn dev
```

App runs at: [http://localhost:5173](http://localhost:5173)

## 📁 Notable Components

- `App.jsx` – Upload & main routing
- `ChapterList.jsx` – Lists extracted chapters
- `QuizViewer.jsx` – Displays generated quiz UI
